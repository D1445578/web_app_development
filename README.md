## 🌟 系統簡介

在校園內，師生經常發生遺失物品的狀況，而拾獲者往往不知如何尋找失主，或是只能交還給生輔組，缺乏一個即時、透明且一目了然的資訊共享平台。

本系統為了解決這個痛點，提供了一個**開放且直覺的遺失物/拾獲物佈告欄**。
- **目標使用者**：常常把東西弄不見的學生、以及在校園內熱心撿到物品的人。
- **解決方案**：透過簡單的表單登記與照片上傳功能，讓遺失物資訊可以快速曝光在首頁，並提供系統的自動比對建議，大幅提高失物尋回效率。

---

## ✨ 核心功能

1. **雙向佈告欄**：明確區分「遺失物 (找東西)」與「拾獲物 (撿東西)」兩個專屬曝光區塊。
2. **直覺的登記體驗**：提供防呆表單，支援照片上傳與聯絡方式填寫。
3. **搜尋與篩選**：可透過地點、物品名稱等關鍵字，快速在大量物件中篩選。
4. **系統比對建議**：點進物件詳情後，系統會反向搜尋資料庫，自動推薦可能有潛在關聯的配對物件。

## 🛠 技術棧

- **前端：** HTML5, CSS3, Bootstrap 5, Jinja2 模板引擎
- **後端：** Python 3.15, Flask 3.1
- **資料庫：** SQLite3
- **架構設計：** 類 MVC 架構 (Models / Routes / Templates)
